При выполнении задания использовались следующие инструменты:

    1) Webpack

        На гит я залью весь проект, собранный будет лежать в папке dist(папки node_modules не будет -> (npm i) please :) )
        В Webpack использовались следующие плагины:

        1.1. HTMLWebpackPlugin
        1.2. CleanWebpackPlugin

        Применялись file-loader и css-loader.

    2) Owl Carousel 2

        Использовал данный слайдер т.к никогда до этого с ним не работал, для себя было интересно попробовать.

        P.S. Второй слайдер получился кривой/косой и не успел сделать кастомную навигацию как на макете, потому что 1 день
        на выполнение задания я просто пропустил, прошу понять и простить).

    3) Jquery

        Jquery использовал лишь для работы со слайдером, всю остальную работу делал на "нативном" Js.

    4) Google Maps API

        Из двух API я решил выбрать Google. Однако сделано в задании не до конца так как по макету. Это обусловлено ткм
        что Google решил сделать свою карту платной, от чего что бы получить полный доступ к карте и ее функционалу надо
        регистрироваться на их сервисе, указывать свои данные, данные карточки. Что бы доработать карту необходимо просто
        в настройках указать координаты указателя и будет точно как по макету.

    5) LESS

        Все стили я писал на Less, и сделал автоматическую конвертацию в CSS (хотя что там делать, WebStorm сам предложил,
        а я согласился).
